# Lab 5: Domain Join and Group Policy Management 🔐

## 📋 Описание
Лабораторная работа по подключению рабочей станции к домену Active Directory, установке компонентов администрирования (RSAT) и настройке групповых политик (GPO) для централизованного управления пользователями и компьютерами.

## 🎯 Цели
1. Подключить компьютер с Windows 10 к домену Active Directory
2. Установить компоненты удалённого администрирования (RSAT)
3. Создать и настроить учётную запись пользователя в домене
4. Настроить групповые политики для:
   - Отключения автоматических обновлений
   - Изменения обоев рабочего стола для всех пользователей
   - Централизованной установки ПО (Mozilla Firefox)
5. Проверить применение политик через `rsop.msc` и `gpresult`

### Задание 1: Подключение компьютера к домену

**Шаги:**
- Открыты свойства системы → «Изменить параметры» → «Изменить»
