# Lab 13: Apache Web Server with HTTPS on Astra Linux 🌐🔐

## 📋 Описание
Лабораторная работа по развёртыванию и настройке веб-сервера Apache2 на отечественной ОС Astra Linux. Практика охватывает установку пакета, проверку работоспособности, включение модуля шифрования (SSL), редактирование конфигурационного файла и настройку виртуального хоста для работы по HTTPS.

## 🎯 Цели
1. Установить Apache2 через пакетный менеджер `apt`
2. Проверить работоспособность сервера через браузер (локально и с хоста)
3. Включить модуль `ssl` для поддержки HTTPS
4. Отредактировать `/etc/apache2/apache2.conf` для настройки VirtualHost на порту 443
5. Настроить пути к сертификату и приватному ключу
6. Перезапустить службу и проверить доступ по `https://`
7. Зафиксировать все этапы скриншотами с ФИО, датой, временем и номером ВМ

## 🛠 Выполненные задания

### Этап 1: Установка Apache2
```bash
# Обновление репозиториев
sudo apt update

# Установка веб-сервера
sudo apt install apache2 -y

# Запуск и добавление в автозагрузку
sudo systemctl enable --now apache2

# Проверка статуса
sudo systemctl status apache2
