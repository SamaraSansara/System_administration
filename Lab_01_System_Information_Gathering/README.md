# Lab 1: System Information Gathering 🔍

## 📋 Описание
Лабораторная работа по сбору системной информации с использованием различных методов и инструментов.

## 🎯 Цель
Научиться получать информацию о системе (имя компьютера, ОС, процессор, RAM, IP-адреса, DNS, пользователи) различными способами.

## 🛠 Использованные методы

### Метод 1: Command Prompt (cmd)
- **Инструменты:** `systeminfo`, `ipconfig`, `hostname`
- **Преимущества:** Быстрый доступ, не требует прав администратора
- **Недостатки:** Ограниченная информация

### Метод 2: MSInfo32
- **Инструменты:** `msinfo32.exe`
- **Преимущества:** Подробная информация в удобном GUI
- **Недостатки:** Требует графический интерфейс

### Метод 3: WMIC / PowerShell
- **Инструменты:** `wmic`, PowerShell cmdlets
- **Преимущества:** Гибкость, автоматизация
- **Недостатки:** Требует изучение синтаксиса

### Метод 4: PowerShell
- **Инструменты:** `Get-ComputerInfo`, `Get-NetIPAddress`
- **Преимущества:** Мощный инструмент, скриптинг
- **Недостатки:** Крутая кривая обучения

### Метод 5: Agent Software
- **Инструменты:** Специализированное ПО для мониторинга
- **Преимущества:** Централизованный сбор, автоматизация
- **Недостатки:** Требует установка дополнительного ПО

## 🔑 Ключевые выводы
- **PowerShell** — наиболее мощный и гибкий инструмент для автоматизации
- **MSInfo32** — лучший выбор для быстрого получения полной информации
- **cmd** — подходит для базовой информации без дополнительных инструментов
- **Agent software** — оптимален для централизованного управления в enterprise-среде
