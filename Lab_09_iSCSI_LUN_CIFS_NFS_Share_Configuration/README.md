# Lab 9: iSCSI, LUN, CIFS/SMB & NFS Share Configuration in Astra Linux 🗄️

## 📋 Описание
Лабораторная работа по настройке сетевых хранилищ в Astra Linux. Практика охватывает создание LUN, настройку iSCSI-таргета и инициатора, маппинг томов, а также организацию общего доступа к файлам через протоколы CIFS/SMB (Windows-совместимый) и NFS (Unix-совместимый).

## 🎯 Цели
1. Создать LUN (Logical Unit Number) в системе хранения
2. Настроить iSCSI-таргет (target) и инициатор (initiator)
3. Выполнить маппинг LUN на Host (хост-систему)
4. Подключить iSCSI-том с Windows-клиента и инициализировать диск
5. Создать файловую систему на новом томе
6. Настроить общий доступ через CIFS/SMB (для Windows-клиентов)
7. Настроить общий доступ через NFS (для Linux-клиентов)
8. Проверить доступ к сетевым ресурсам с проводника Windows

## 🛠 Выполненные задания

### Задание 1: Создание LUN (Logical Unit Number)
- Создан LUN через интерфейс управления хранилищем (targetcli / web-интерфейс)
- Задан размер тома, тип (thin/thick provisioning)
- Присвоено уникальное имя/идентификатор LUN

### Задание 2: Создание iSCSI инициатора
- Установлен пакет `open-iscsi` (если не был предустановлен)
- Сконфигурирован `/etc/iscsi/iscsid.conf` (IQN инициатора, параметры аутентификации)
- Запущена служба: `sudo systemctl enable --now iscsid`

### Задание 3: Создание Host (хоста-клиента)
- Зарегистрирован Host в системе хранения по IQN (iSCSI Qualified Name)
- Настроены права доступа (ACL) для хоста

### Задание 4: Маппинг LUN на Host
- Выполнено связывание созданного LUN с зарегистрированным Host
- Проверена видимость LUN с стороны инициатора: `sudo iscsiadm -m session -P 3`

### Задание 5: Подключение iSCSI в Windows
- Открыт `iSCSI Initiator` в Панели управления
- Введён IP-адрес таргета, выполнено быстрое подключение
- Новый диск отображается в `diskmgmt.msc` как "Не проинициализирован"

### Задание 6: Инициализация тома в Управлении дисками
- Инициализирован диск (GPT/MBR)
- Создан простой том, отформатирован в NTFS/exFAT
- Присвоена буква диска, проверена доступность в Проводнике
