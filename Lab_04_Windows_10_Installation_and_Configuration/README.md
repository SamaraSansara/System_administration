## 📊 Ключевые выводы

### Этапы установки Windows 10

**1. Подготовка:**
- Выбор редакции ОС (Home/Pro/Enterprise)
- Проверка совместимости оборудования
- Подготовка драйверов (особенно для VirtIO в виртуальных машинах)

**2. Разметка диска:**
- Создание/изменение разделов
- Форматирование (удаление всех данных)
- Выбор типа разметки (MBR/GPT)

**3. Настройка системы:**
- Региональные стандарты
- Язык и раскладка клавиатуры
- Часовой пояс

**4. Учетная запись:**
- Локальная учетная запись vs Microsoft Account
- Настройка пароля и контрольных вопросов
- Права доступа (Admin/Standard User)

**5. Драйверы:**
- Критически важны для работы оборудования
- VirtIO драйверы необходимы для виртуальных машин KVM/QEMU
- Установка через диспетчер устройств или setup-утилиты

### VirtIO драйверы

**Назначение:**
- Оптимизация работы виртуальных машин
- Улучшение производительности диска и сети
- Паравертуализация вместо полной эмуляции

**Основные компоненты:**
- **virtio-scsi** — дисковый контроллер
- **virtio-net** — сетевой адаптер
- **virtio-balloon** — управление памятью
- **qxl** — графический драйвер

### Безопасность

**Пароль:**
- Минимум 8 символов
- Комбинация букв, цифр, спецсимволов
- Уникальность (не использовать одинаковые пароли)

**Контрольные вопросы:**
- Альтернативный способ восстановления
- Ответы должны быть запоминающимися
- Не использовать общедоступную информацию

## 📚 Теоретическая база

### Установка операционной системы

**Типы установки:**
1. **Clean Install** — чистая установка с форматированием
2. **Upgrade** — обновление с сохранением данных
3. **Dual Boot** — установка второй ОС

**Разметка диска:**
- **MBR (Master Boot Record)** — до 2 ТБ, до 4 основных разделов
- **GPT (GUID Partition Table)** — до 9.4 ЗБ, до 128 разделов

### Драйверы устройств

**Классификация:**
- **PnP (Plug and Play)** — автоматическая установка
- **Legacy** — ручная установка
- **Signed** — подписанные драйверы
- **Unsigned** — неподписанные (требуют отключения проверки)

**Источники:**
- Windows Update
- Сайт производителя
- Поставляемый ISO-образ
