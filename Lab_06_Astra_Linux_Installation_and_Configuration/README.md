# Lab 6: Astra Linux Installation and Basic Configuration 🐧

## 📋 Описание
Лабораторная работа по установке отечественной операционной системы Astra Linux, настройке сетевого интерфейса, установке гостевых дополнений и базовой конфигурации системы.

## 🎯 Цели
1. Освоить процесс установки Astra Linux (Special Edition / Common Edition)
2. Настроить сетевой интерфейс через `/etc/network/interfaces`
3. Установить и настроить загрузчик GRUB
4. Установить `qemu-guest-agent` для интеграции с гипервизором
5. Проверить работоспособность системы после перезагрузки

## 🛠 Выполненные задания

### Задание 1: Установка Astra Linux

#### 1.1 Настройка имени компьютера
**Выполненные действия:**
- Задано имя хоста: `[hostname]`
- Проверено разрешение имени через `hostname` и `hostname -f`

#### 1.2 Создание учётной записи администратора
**Параметры учётной записи:**
- Имя пользователя: `admin` (или заданное)
- Пароль: установлен с требованиями сложности
- Добавление в группу `sudo` для получения привилегий

#### 1.3 Выбор программного обеспечения
**Установленные компоненты:**
- Базовая система (base system)
- SSH-сервер (openssh-server) — для удалённого управления
- Утилиты командной строки (curl, wget, net-tools)
- Графическая среда (опционально: Fly DE / KDE / XFCE)

#### 1.4 Установка загрузчика GRUB
**Выполненные действия:**
- Выбран целевой диск для установки GRUB: `/dev/vda` (или `/dev/sda`)
- GRUB установлен в MBR диска
- Проверена загрузка системы после установки
